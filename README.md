## 路由器设置  
1.	将外网线连接至路由器WAN口（一般是颜色不一样 单独的那个口）  
2.	将路由器上网方式设置为DHCP（自动获取）  
3.	查看路由器WAN口是否正确获取到IP （100.64.XXX.XXX） 
4.  电脑连接到路由器的WIFI或网口，不是连FeiYoung热点
5.  可以使用LinkGoYoung进行连接了
